Algorithm Explanation

Deep Q-Learning: The AI agent learns optimal routing paths through experience. It observes network states (congestion, failures) and learns which actions (route selections) lead to better performance over time.

Traditional Routing: Uses shortest path algorithms like Dijkstra, which always choose the path with minimum hops or distance, regardless of current network conditions.

Current Routing Decision

Select source and destination nodes to see routing decisions.

Q-Learning Information

Q-Values: Represent the expected future reward for taking a specific action in a given state. Higher Q-values indicate better routing choices.

Learning Process: The agent explores different paths, observes results, and updates its knowledge to make better decisions in similar situations.

Adaptation: As network conditions change, the agent continuously learns and adapts its routing strategy.

How to Use the Simulator

  1. Select source and destination nodes from the dropdowns
  2. Choose between AI routing or traditional shortest path
  3. Start the simulation to see packet routing in action
  4. Adjust network conditions to see how algorithms adapt
  5. Compare performance metrics between different approaches

Educational Purpose: This simulator demonstrates how AI can improve network routing by learning from experience and adapting to changing conditions.
